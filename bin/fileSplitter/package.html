File Splitter & Stitcher

This package provides a GUI to split and stitch files. It can also Compress or Encrypt the chosen files.

Usage

Click on "Aggiungi File" to choose files that you want to split.

You can also choose control files (.partitioninfo) to stitch split files together.

You can remove all selected files from the queue by clicking on "Reset Coda".

If you want to split a file, click on the "Split Files" button.

If you selected a control file and you want to stitch back together its parts, click the "Ricomponi File" button.

You can customize how the file gets split by clicking on one of the four options at the bottom of the left panel.

Don't forget to also give a input value in the text field. It is used as a parameter and only accepts positive integers.

On the right panel, you can choose an output folder for the splitting and stitching by clicking on the "Scegli" Button.
